# Notification System Design

## Scope

This repository is organized for the campus notifications evaluation track.

## Frontend

The React/Vite frontend lives in `notification_app_fe`.

## Backend

The dedicated backend folder is `notification_app_be`.

## Logging Middleware

The logging middleware folder is `logging_middleware`.

## Notes

- `node_modules` is ignored in `.gitignore`.
- The current app uses the notification API and structured logging middleware.
